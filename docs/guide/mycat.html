花花是一只猫

影大宝2022-06-08 00:00:00

INFO

  1. 花花是一只猫，此篇以第一人称讲述;
  2. 此篇博文实际是花花口述 略懂喵语的影大宝记录。

# 本喵出生之时，夏末初秋。


没错，左边那位就是本喵。

# 本喵毕业之时，学富五车。


本喵善于思索、有见识、具有文人气质。

# 本喵初入社会，不幸被绝育。


纵使被诱惑千百遍，我依然心如止水。——本喵窥见自夏目漱石《我是猫》

# 本喵光临影舍，蓬荜生辉。


影舍指影大宝的房子，本喵光临至此，柴门有庆。

# 本喵与狮子同属喵科，威风凌凌。


仰天长啸出门去,我辈岂是蓬蒿喵。

# 本喵乃大橘类目，天高马肥，老饕喵王。


本喵授课：天高马肥的谚语来自古诗“云净妖星落，秋深赛马肥”。

# 今日本喵自传就写到这里，来日方长。
